### Core Functionalities

#### Admin Control Center
- **User Management**: Create and manage staff accounts (Librarians).
- **System Audit Logs**: Real-time monitoring of critical system actions and security events.
- **Financial Overview**: Track library revenue and transaction history.
- **System Status**: Monitor the health and connectivity of all microservices.

#### Librarian Dashboard
- **Inventory Management**: Full CRUD operations for the book catalog.
- **Loan Processing**: Manage active loans, track overdue items, and process book returns.
- **User Search**: Quickly find readers in the system to assign loans or check history.
- **Transaction Tracking**: Monitor fine payments and financial logs related to loans.

#### Reader Experience (Mobile App)
- **Digital Catalog**: Browse the entire library collection with real-time availability.
- **My Loans**: Track personal reading history and upcoming return deadlines.
- **Instant Search**: Find books by title, author, or category.
- **Account Management**: View profile details and digital library card.
